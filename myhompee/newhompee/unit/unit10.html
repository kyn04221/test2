10단원 - CSS 고급 선택자

연결 선택자

  • 하위 선택자 : eciton p {...}

    • 부모요소 section 안에 포함함된 모든 하위 요소에 스타일이 적용됨

  • 자식 선택자 : section > p {...} 일때

    • 요소1과 요소2 사이에 '>' 기호 사용

    부모요소인 section 안에 포함된 p요소중 자식 p 요소에만 스타일 적용됨

  • 인접 형제 선택자 : h1 + p {...}

    • 같은 부모를 가진 형제 요소 중 첫번째 동생 요소에만 스타일 적용

    요소1과 요소2 사이에 '+' 기호 사용

  • 형제 연산자 : h1 ~ p{...}

    • 모든 형제요소에 다 적용됨

속성 선택자

  • 지정한 속성을 가진 요소를 찾아 적용
  • a요소중 href속성값이 있는 요소만 적용

    • a[href]{...}

    <a href="#"> 가나다 </a>

  • [속성 = 값] : 속성과 속성값이 일치하는 요소를 찾아 적용

    • a [target="_blank"] : a 요소중 target 속성의 값이 _blank인 요소에만 적용

  • [속성~=값] : 여러 속성값 중 해당 값이 포함되어 있는 요소를 찾자 적용

    • a [class ~="button"] : {...} 아래 모두 포함

    <a href="#" class="button">메뉴 3</a>

    <a href="#" class="flat button">메뉴 3</a>

  • [속성 = 값] : 속성과 속성 값이 일치하는 요소를 찾아 스타일 적용 하이픈 가능

    • a [title |= us]{...}

    <a href="#" title="us-english">영어</a>

  • [속성^=값] : 특정 값으로 시작하는 속성을 가진 요소를 찾아 스타일 적용

    • a [title ^="eng"] {...}

    <a href="#" title="english">영어</a>

  • [속성 $= 값] : 특정 값으로 끝나는 속성을 가진 요소를 찾아 스타일 적용

    • a [title $="xls"] {...}

    <a href="intro.xls">엑셀파일</a>

  • [속성*=값] : 값의 일부가 일치하는 속성을 가진 요소를 찾아 스타일 적용

    • a[href *= "w3"] {...}

    <a href="https://www.w3.org/TR/css3-mediaqueries>미디어쿼리</a>/p>

가상 클래스

  • 순서 중요 :link -> :visited -> :hover -> :active
  • :link - 방문하지 않은 링크에 스타일 적용
  • :visited - 방문한 링크에 스타일 적용
  • :active - 웹 요소를 활성화했을 때의 스타일 적용
  • :hover - 웹 요소에 마우스 커서를 올려놓을 때의 스타일 적용
  • :focus - 웹 요소에 초점이 맞추어졌을 때의 스타일 적용

  • 요소 상태에 따른 가상 클래스 선택자

    • :target - 앵커로 연결된 부분에 스타일 적용

    :enabled, :disabled – 요소의 사용 여부에 따라 스타일

    :checked – 라디오 버튼이나 체크 박스에 체크했을 때 스타일 적용

    :not – 특정 요소를 제외하고 스타일 적용

  • 구조에 따른 가상 클래스

    • :only-child --> 부모 안에 자식 요소가 하나뿐일 때 자식 요소를 선택합니다.

    A: only-type-of --> 부모 안에 A 요소가 하나뿐일 때 선택합니다.

    first-child --> 부모 안에 있는 모든 요소 중에서 첫 번째 자식 요소를 선택합니다

    : last-child --> 부모 안에 있는 모든 요소 중에서 마지막 자식 요소를 선택합니다

    A: first-of-type --> 부모 안에 있는 A 요소 중에서 첫 번째 요소를 선택합니다.

    A: last-of-type --> 부모 안에 있는 A 요소 중에서 마지막 요소를 선택합니다.

    inth-child(n) --> 부모 안에 있는 모든 요소 중에서 n번째 자식 요소를 선택합니다.

    inth-last-child(n) --> 부모 안에 있는 모든 요소 중에서 끝에서 n번째 자식 요소를 선택합니다.

    A:nth-of-type(n) --> 부모 안에 있는 A 요소 중에서 n번째 요소를 선택합니다.

가상 요소

  • 화면 꾸미기용 요소를 웹 문서에 포함시키지 않기 위해 가상 요소 사용

    • ::first-line : 특정 요소의 첫번째 줄에 스타일 적용

    ::first-letter : 특정 요소의 첫번째 글자에 스타일 적용

    ::before : 특정 요소의 앞에 지정한 콘텐츠 추가

    ::after : 특정 요소의 뒤에 지정한 콘텐츠 추가
