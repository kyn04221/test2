javascript

외부 스크립트 파일 연결해서 작성하기

  • <script src="외부스크립트 경로"></script>
  • 식

    • >> 값을 만들어 낼 수있다면 모두 식이될수 있다

    >> 식은 변수에 저장된다

  • 문

    • >> 문의 끝엔 세미콜론 ; 을 붙여서 구분

    >> 넓은 의미에서 식이나 값이 포함됨

입출력방법

  • alert("hi")

    • >> '확인'버튼이 있는 메시지 창 표시

  • confirm("hi")

    • >> '확인'과 '취소'버튼이 있는 창 표시

  • var name = prompt("이름을 입력하세요", "가나다")

    • >> 텍스트 필드가 있는 창 표시

    >> 사용자 입력값을 가져와 프로그램에서 사용

  • document.srite()

    • >> document에서 괄호 안에 내용을 표시(write)하는 명령문

    >> 괄호 안에서 큰따옴표나 작은 따옴표 사용

    >> 따옴표 안에는 html태그도 사용 가능

  • console.log()

    • >> 괄호 안에 내용을 콘솔창에 표시

    >> 괄호 안에 변수가 들어갈 수도 있고, 따옴표 안에 텍스트를 넣을수 있음
