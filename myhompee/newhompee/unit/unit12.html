반응형 웹

뷰포트

  • vw : 1vw는 뷰포트 너비의 1%와 같다
  • vh : 1vh는 뷰포트 높이의 1%와 같다
  • vmin : 뷰포트의 너비와 높이중에서 작은 값의 1%와 같다
  • vmax : 뷰포트의 너비와 높이중에서 큰 값의 1%와 같다

미디어 쿼리

  • 미디어 쿼리 : 접속하는 장치에 따라 특정한 css 스타일을 사용
  • @media [only | not ] : 미디어 유형 [and 조건] * [and 조건]

    • >> only : 미디어 쿼리를 지원하지 않는 웹 부라우저에서 미디어 쿼리를 무시하고 실행하지 않음

    >> not : not 다음에 지정하는 미디어 유형은 제외

    >> and : 조건을 여러개 연결해서 추가할수 있음

    >> @media screen and (min-whdth: 760x) and (max-width:1489) {...}

그리드 레이아웃 (1차원)

  • 플렉스 박스 레이아웃

    • >> 수평이나 수직 중 하나를 주축으로 정하고 박스를 배치

    >> 여유공간이 생길경우 너비나 높이를 적절하게 늘리거나 줄일수 있음

  • css 그리드 레이아웃

    • >> 수평이나 수직 어디로든 배치 가능

    >> 요소를 레고블록 끼워 맞추듯 배치 가능

플랙스 박스 레이아웃

  • 플렉스 박스 레이아웃

    • 그리드 레이아웃을 기본으로, 플렉스 박스를 원하는 위치에 배치 함

    여유 공간에 따라 너비나 높이, 위치를 자유롭게 변형할 수 있음

  • display : 배치 요소들을 감싸는 부모요소를 플렉스 컨테이너로 지정

    • >> flex : 컨테이너 안의 플렉스 항목을 블록 레벨 요소로 배치함\

    >> inline-flex : 컨테이너 안의 플렉스 항목을 인라인 레벨 요소로 배치

  • flex-direction : 플렉스 항목의 배치를 위해 주축과 방향 지정

    • >> row : 가로, 왼쪽에서 오른쪽으로 배치 (기본값)

    >> row-reverse : 가로, 오른쪽에서 왼쪽 배치

    >> column : 세로, 위쪽에서 아래쪽으로 배치

    >> column-reverse : 세로, 아래쪽에서 위쪽으로 배치

  • flex-wrap : 플렉스 항목을 한줄 또는 여러줄로 배치

    • >> nowrap : 플렉스 항복을 한줄로 표시 (기본값)

    >> wrap : 플렉스 항목을 여러줄에 표시

    >> wrap-reverse : 여러줄이 표시되지만 화면을 줄이면 끝줄부터 위로 올라감

  • justify-content : 플렉스 항목을 주축 방향으로 배치할 때의 배치 기준

    • >> flex-steart : 시작점 배치

    >> flex-end : 끝점 배치

    >> center : 중앙 배치

    >> space-between : 시작점과 끝점에 배치후 그사이 같은 간격 배치

    >> space-around : 모두 같은 간격 배치

  • align-items : 교차축을 기준으로 하는 배치 방법
  • align-self : 교차축에서 특정 항목만 지정하고 싶다면 align-self 사용

    >> flex-steart : 시작점 배치

    >> flex-end : 끝점 배치

    >> center : 중앙 배치

    >> baseline : 문자 기준선에 맞춰 배치

    >> stretch : 플렉스 항목을 늘려 교차축에 가득차게 배치

  • align-content : 플렉스 항목이 여러 줄로 표시될 때 교차 축 기준의 배치 방법 지정

    • >> flex-steart : 시작점 배치

    >> flex-end : 끝점 배치

    >> center : 중앙 배치

    >> space-between : 시작점과 끝점에 배치후 그사이 같은 간격 배치

    >> space-around : 모두 같은 간격 배치

    >> stretch : 플렉스 항목을 늘려 교차축에 가득차게 배치

css 그리드 레이아웃 (2차원)

  • 플렉스 박스 레이아웃은 주축/교차축 개념이 있지만, CSS 그리드 레이아웃은 양쪽 방향 모두 사용
  • 가로(row)와 세로(column)로 화면을 구성하고, 줄과 칼럼 사이의 여백을 조절
  • display : 배치 요소들을 감싸는 부모 요소를 그리드 컨테이너로 지정

    • >> gird : 컨테이너속 항목을 블록으로 배치

    >> inline-grid : 컨테이너속 항목을 인라인으로 배치

  • grid-template-columns : 칼럼 개수와 너빗값
  • grid-template-rows: 줄 개수와 너빗값
  • 상대적인 크기를 지정하는 fr 단위

    • >> 반응형 웹 디자인을할땐 상대적인 크기를 지정하는 fr(fraction) 단위 사용

    >> 너비 2:1:2 배치시 --> grid-template-columns: 2fr 1fr 2fr;

  • 값이 반복된다면 repeat( ) 함수

    • grid-template-columns: 1fr 1fr 1fr; --> grid-template-columns: repeat(3, 1fr)

  • auto-fill, auto-fit : 화면 너비에 따라 칼럼 개수를 조절음

    • >> auto-fit : 남는 공간 없이 꽉 채우기

    >> auto-fill : 칼럼의 최소 너비까지만 표시하고 남는 공간은 그대로 둠

  • 그리드 라인

    • 레이아웃에는 눈에 보이지 않는 그리드 라인이 포함되어 있어 그리드 항목을 배치할 수 있음

    >> grid-column-start: 번호 : 칼럼 시작 라인의 번호 지정

    >> grid-column-end: 번호 : 칼럼 마지막 라인 번호 지정

    >> grid-column: 번호/번호 : 칼럼 시작과 끝 번호를 넣어 사용

    >> grid-end-start: 번호 : 줄 시작 라인 번호 지정

    >> grid-row-end: 번호 : 줄 마지막 라인 번호 지정

    >> grid-row: 번호/번호 : 줄 시작과 끝번호 넣어 사용

  • 템플릿 영역

    • grid-area: 템플릿 영역을 만든 후 배치
